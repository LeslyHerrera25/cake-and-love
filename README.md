# 🎂 Cake & Love - Sistema de Pastelería

Sistema web desarrollado para la gestión de una pastelería, permitiendo la visualización de productos, carrito de compras, pedidos y autenticación de usuarios.

## 🚀 Funcionalidades
- Catálogo de productos
- Carrito de compras
- Sistema de pedidos
- Inicio de sesión de usuarios
- Personalización de productos
- Reportes

## 🛠️ Tecnologías utilizadas
- Python
- Django
- HTML
- CSS
- SQLite
